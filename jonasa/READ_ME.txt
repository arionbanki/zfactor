Þessi lausn er skrifuð í Rust.

Til þess að keyra hana þarf að:
- Installa Rust (https://www.rust-lang.org/en-US/install.html)
- Installa C++ Build Tools. (https://www.visualstudio.com/downloads/#build-tools-for-visual-studio-2017)
- Opna console og keyra eftirfarandi í rót lausnar:
	- cargo build
	- cargo test
	- cargo run
